> When I tap the 5x button I want to use the 5x camera

The stock camera app does not interpret the user tapping the 5x button as wanting the 5x camera; it interprets it as the user wanting the highest quality photo at 5x magnification that the system (thinks it) can deliver, because there are a lot of potential tradeoffs to be made.

It's a subtle distinction, but it's there. If you want a behavior like you describe, use a "pro" app like Halide or Obscura.
