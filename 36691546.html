Thunderbird really made me appreciate snaps on Ubuntu.

There was a buggy native version released at one point, and the repo was a bit delayed in getting the new version (it's my daily driver so pretty impacting).

I saw the snap version was up to date, and tried it. It worked which was all well and good. But a while later another buggy version came out which made me oauth authenticate repeatedly. A single command reverted the snap, no version pinning or anything.

A lovely way to install these larger complex programs (don't get me wrong, apt install is 98% of my system).

Congrats to the thunderbird team, thanks for making such a great program that makes work that bit more tolerable!!
