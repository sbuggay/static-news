Alphatab is a cross platform library for the parsing, interpreting and drawing of guitar tabs (such as Guitar Pro format files).

I'm a long time fan of Alphatab, which a labor of love for Guitar Tab in general from Daniel Kuschny. If you ever need to build a piece of software that shows guitar tab then this is basically the library to use (across multiple platforms). It's very much not just a javascript thing.

Long may it continue!
