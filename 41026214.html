Great project. I am mostly impressed that Glisp provides a great experience of reversely modifying code by dragging the ui.

Personally, I really agree with the philosophy of code for design. I often feel that code is more useful than gui in many cases. In addition, I have always thought that there should be a better way to make slides. I have tried revealjs and now use remark, but I hope there will be a simpler way to achieve programmatic specification like keynote magic move.
