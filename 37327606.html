Given this is about an Amiga format: How I miss Datatypes.

The datatypes library provided a simple object model that allows you to drop a system-wide library in place to handle a new format.

If you wanted to scale images for thumbnails, for example, you would simply obtain a data type object and get back decoded image data and scale it, and automatically be able to handle everything the user have picture datatypes for.

So you add basic support for a format once.
