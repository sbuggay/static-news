> Please note that you will need the Vortex86 SOM and an AGP card to make use of this motherboard, we offer the SOM as a separate purchase for a discounted price and recommend that you include it in your order before checking out.

> [...]

> Introducing the ITX-Llama, a retro-inspired true x86 motherboard designed for those passionate about late-DOS and early Windows 98 systems. Built using the powerful and versatile Vortex86EX system-on-module (SOM), the ITX-Llama boasts a P6 (Pentium Pro/II/III)-compatible CPU running at 100-500 MHz, with 128 MB of LPDDR3 RAM and an 8 MB ROM. This compact system delivers performance comparable to a Pentium 233, making it ideal for vintage applications and gaming.
