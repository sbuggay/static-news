What's different from running in Sandboxy?

beardog|parent|next|

Windows Sandbox is an official Microsoft feature that uses hardware virtualization for quickly made disposable Windows VMs.

Sandboxie is an OS-level 3rd party app that uses syscall hacks to sandbox applications.

I say use Windows Sandbox if you care about security more than convenience.
