I also know some Hindi if that is a language you speak and can give you better notes, probably, with that language as well. Moving on to the notes:

-"know" is a transitive verb, so it always takes an object of its action. One must know something. When you add the extra "is" next to the verb "know," it indicates the passive, which would take an object, but none are indicated here. You could say "It is known that..." as a passive construction with the "that" beginning a new clause, but in your case it is better just to delete the "is."

-"for" is the preposition for the indirect object in English, so is "to" in many cases. I believe here you confused it with to, which can also be used to indicate an indirect object, but "to" has a number of different usages in English, whereas "for" is only used for indirect objects. It can also be used to form a complimentary infinitive, as we see here. Like, for instance "He eats to help him gain weight."

-"similar" is being used adverbial here, so it should be indicated as such by adding "-ly" to the end of the adjective. English has no cases in the same way as related languages, so its not possible to indicate adverbial usage of a noun or adjective without either transforming it or saying something like "they behave in a similar manner," where "in...manner" indicates the way in which the action is performed, like an adverb does.
